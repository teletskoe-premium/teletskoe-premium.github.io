# 🚤 Teletskoe Premium | Экспедиции с Евгением

Премиальный веб-сервис для бронирования эксклюзивных путешествий по Телецкому озеру (Алтай).

## ✨ Особенности проекта
* **Интерактивный UI:** Уникальная навигация с помощью «катера-переключателя».
* **Детальные карточки:** Иммерсивное описание маршрутов с галереями.
* **Smart-виджеты:** Динамический расчет времени до «Золотого часа» и статус навигации.
* **Mobile First:** Полная оптимизация под смартфоны туристов.

## 🛠 Технологии
* HTML5 / CSS3 (Tailwind CSS)
* JavaScript (Vanilla)
* Lucide Icons & Animate.css
* Deployment: Netlify + GitHub Autodeploy
